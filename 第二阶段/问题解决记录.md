# 一、GitHub 文件夹创建
 
 > GitHub不支持创建空文件夹，所有文件夹必须包含至少1个文件（可新建空的 **.gitkeep**文件占位）

网页端创建文件夹步骤
 
1. 进入GitHub仓库，点击Add file → Create new file

2. 文件名栏输入： 文件夹名/.gitkeep （例： 第一阶段/.gitkeep ）

3. 下拉点击Commit new file，自动生成对应文件夹

# 二、易错点（Git)
 - Git不能Ctrl+v复制，只能双击点Paste/Shift+lns
 - Git bash 里面每一个字母间都有空格
   ```
   bash
   git config --global user.name "池海秀"
   ```
   **config与--global/name与 "池海秀"之间都有空格
